# Snake Game in CPP

A simple terminal- Snake Game written in C++.

## How to Play

- Use **W** to move up  
- Use **S** to move down  
- Use **A** to move left  
- Use **D** to move right  
- Press **Q** to quit the game

Eat the food (`O`) to grow the snake. Avoid hitting the walls (`#`) or your own tail.
